# BG3攻略工具设计文档

## 概述

开发一个《博德之门3》(Baldur's Gate 3) 辅助攻略桌面应用，提供剧情/任务引导功能，支持游戏内实时查看。

## 目标用户

- 中文玩家
- 主要在PC平台(Steam/GOG)游玩
- 需要实时查看任务攻略，不想频繁切换窗口或查网页

## 技术方案

### 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | React + TypeScript |
| 桌面框架 | Tauri (Rust后端) |
| 数据存储 | SQLite |
| OCR引擎 | Tesseract.js |
| UI组件 | TailwindCSS + shadcn/ui |

### 架构设计

```
┌─────────────────────────────────────┐
│           Tauri 应用                 │
├─────────────────────────────────────┤
│  前端 (React + TypeScript)          │
│  ├── 主窗口 (攻略浏览)               │
│  ├── Overlay窗口 (游戏内悬浮)        │
│  └── 设置窗口                        │
├─────────────────────────────────────┤
│  后端 (Rust)                         │
│  ├── SQLite数据存储                  │
│  ├── OCR识别 (调用Tesseract.js)      │
│  ├── Overlay渲染                     │
│  └── 数据导入/更新                    │
└─────────────────────────────────────┘
```

## 核心功能

### 1. 任务百科

- 按章节浏览所有任务
- 按类型筛选：主线/支线/同伴任务
- 任务搜索功能

### 2. 任务详情

- 分步骤指引
- 分支选择提示，显示各选项后果
- 任务位置、奖励信息
- 相关NPC/地点链接

### 3. 进度追踪

- 手动勾选已完成的任务/步骤
- 截图识别：捕获游戏任务日志，自动匹配当前任务
- 进度统计：已完成任务数、当前章节

### 4. 游戏内Overlay

- 悬浮窗显示当前任务提示
- 无边框、透明背景、始终置顶
- 可拖拽定位、调整大小
- 热键操作：
  - `Ctrl+G` 显示/隐藏
  - `Ctrl+←/→` 上一步/下一步

### 5. 截图识别

- 自动捕获游戏窗口截图
- 或用户手动粘贴截图
- OCR识别任务日志文字
- 匹配数据库定位当前任务
- 提供手动修正选项（识别不准确时）

## 数据模型

### 章节 (Chapter)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| name | TEXT | 章节名称 |
| order | INTEGER | 排序序号 |

### 任务 (Quest)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| chapter_id | INTEGER | 所属章节 |
| name | TEXT | 任务名称 |
| type | TEXT | 类型：main/side/companion |
| description | TEXT | 任务描述 |
| prerequisites | TEXT | 前置任务ID列表(JSON) |

### 任务步骤 (QuestStep)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| quest_id | INTEGER | 所属任务 |
| order | INTEGER | 步骤序号 |
| description | TEXT | 步骤描述 |
| location | TEXT | 地点名称 |
| rewards | TEXT | 奖励列表(JSON) |

### 选择分支 (Choice)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| step_id | INTEGER | 所属步骤 |
| description | TEXT | 选择描述 |
| consequences | TEXT | 后果描述(JSON) |
| leads_to_step_id | INTEGER | 选择后跳转步骤 |

## UI设计

### 主界面

- 左侧：章节列表 + 任务类型筛选
- 右侧：任务详情卡片，分步骤展示
- 底部：进度统计栏
- 顶部：搜索框 + 设置按钮

### Overlay界面

- 小窗口(~300x200px)
- 显示当前任务名称、当前步骤、位置提示
- 分支选项简要提示
- 上一步/下一步导航按钮

## 数据来源

爬取中文攻略站数据：
- 3DM攻略
- 游民星空攻略
- 百度贴吧精华帖

数据整理为JSON格式导入应用。

## 数据更新
